Java 8 Date Time Explained Series - Part 1

Jun 11, 2019
Tags: java java8 date-time
Categories: java dev
Background Java has introduced improved date time APIs in Java 8 release by incorporating joda-time. These new APIs under java.time package. java.time package contains APIs for dates, times, instants, time-zones and durations. There are 4 categories in these APIs Instants (a point in time since 1970) Local date time (a date & time local to the consumer of the API) Time zone date time (A date time in given time zone) General purpose APIs like Clock, Duration, Period etc… In this part-1 of date time series we will conver the Instant.
